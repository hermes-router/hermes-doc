dispatch.send module¶

send.py¶

The file for sending the dicoms to target destination.

dispatch.send.execute(source_folder, success_folder, error_folder)[source]¶

Execute the dcmsend command. It will create a .lock file to indicate that the folder is being sent. This is to prevent double sending. If there happens any error the .lock file is deleted and an .error file is created. Folder with .error files are _not_ ready for sending.
